es6 的 class

我们都知道es6 中的 class 基本就是function的语法糖.

所谓的静态属性,方法,其实就是直接往构造函数上定义.

属性定义在实例上，方法定义在原型上。

装饰器

es7有个提案，就是装饰器。用来修饰类的行为。装饰器本质就是个函数

在typescript上装饰器可以用来

  • 修饰类,就是修饰构造函数，
  • 修饰静态变量，也是修饰构造函数,(静态变量是定义在构造函数上的)
  • 修饰静态方法, 同上.
  • 修饰实例变量,就是修饰原型对象.
  • 修饰实例方法,就是修饰原型对象.
  • 修饰方法参数,也是修饰原型对象

装饰器实际是利用 Object.defineProperty 对属性,方法进行拦截,重新定义修改而已.

可以看出我们可以重新定义任何属性,方法,不限于prototype.constructor,Function.name等等一些有趣的东西.

装饰器可以有参数,无非就是闭包,返回个函数而已.

元数据与反射

元数据

typescript 里,可以打开元数据选项,打开后可以使用Reflect.getMetadata方法,获取类的类型,类的方法属性类型,类的方法返回类型.(获取不到装饰器的参数和装饰器名字滴)

反射

上面我们分析过了,装饰器就是函数,让我们可以自由定义属性,方法,所以这些属性,方法必须由我们亲自存储, 再获取.

而reflect-metadata这个库就可以帮我们快速存储,获取的.
